# ADMIN LANDING PAGE - TRANG CHÍNH / CARD DASHBOARD

Access từ trang admin web tamsonyachting.com

## SECTION 1 QUICK STATS (OVERVIEW)

- Total Cards (tương ứng với total Card owners)
  - (VD: 28 Cards)
- Active Cards (Số card đã kích hoạt thành công)
  - (VD: 24 Active)
- Newest Bookings (Booking <Tên du thuyền> gần đây, bởi Card ID nào)
  - (VD: Azimut S7 · 2h ago · Card #0181)

INTERACTION:
- Total / Active -> Card list (với mobile: Hiển thị dạng card dọc)
- Newest Bookings -> Booking detail (với mobile: Tap 1 chạm vào chi tiết)

## SECTION 2 PRIMARY ACTIONS (CTA)

CTA dạng grid icon + label

- Update Charter Pricing -> Chỉnh giá charter
- View / Edit / Update Member list (tên, sđt chủ thẻ & khách mượn thẻ)
- Card Status Management: add / edit / delete card
- Post News / Notices
- CTA dùng icon + label, ưu tiên thao tác 1 chạm trên mobile

## SECTION 3 RECENT ACTIVITY

Hiển thị các sự kiện diễn ra gần đây, trong 3 ngày:
- New member added
- New booking
- Card status updated

UI:
- Dạng list / timeline
- Có thể scroll
- Tap vào reference để xem chi tiết

Format: Time · Action · (Detailed info) · #Card ID
- VD: 2h ago · Booking created · Azimut S7 · Card ID #0231
- VD: 1 day ago · Card activated · Card ID #0181
- VD: 3 days ago · Card upgraded · Card ID #0264

## SECTION NAVIGATION ENTRY (MODULES)

(Menu chính - dạng grid hoặc sidebar tùy layout)

- Members
- Benefits
- Reports
- Settings

Icon rõ ràng, label ngắn, nhất quán
