# Features and User Stories (Rally-style)

Quick links: [Summary](./summary.md) · [Tech Stack](./tech-stack.md)

## Public Landing (SEO)

- US001 — Access landing page: As a visitor, I can access `/voyageplus` from search without seeing private data.
- US002 — See public welcome: As a visitor, I see a welcome page that explains private access.
- US003 — Switch language: As a visitor, I can switch VI/EN on the landing page.

## Entry / Welcome Overlay

- US004 — Block direct access: As a visitor, a full-screen overlay prevents direct access to content.
- US005 — Display branding: As a visitor, I see logo, headline, subtext, and CTA.
- US006 — Show error state: As a visitor, I see a light error message when access is invalid.

## QR Entry Flow

- US007 — Open from QR: As a member, I can scan QR on my card to open the landing page.

## Serial Verification

- US008 — Verify serial: As a member, I can enter my card serial number to verify access.
- US009 — Show status: As a member, I see status Valid/Invalid/Active after submission.
- US010 — Contact admin: As a member, I can contact admin if my serial is invalid.

## Member Portal – Card Overview

- US011 — View card status: As a member, I can see card name and status (Base/Gold/Diamond).
- US012 — View usage summary: As a member, I can see voyages completed and spending progress.
- US013 — View last network booking: As a member, I can see last network booking details.

## Member Portal – Charter Pricing

- US014 — View pricing tiers: As a member, I see yacht pricing by card level.
- US015 — Compare prices: As a member, I see pre- and post-discount prices.

## Member Portal – Benefits

- US016 — View current benefits: As a member, I can see benefits for my level.
- US017 — Preview locked benefits: As a member, I can preview locked future benefits.

## Member Portal – Voyage History

- US018 — View trips: As a member, I can see trip list with date, yacht, route, type, and status.
- US019 — View trip savings: As a member, I see savings for completed trips only.

## Member Portal – Progress & Next Level

- US020 — See upgrade gap: As a member, I see how much more to reach the next level.
- US021 — Prompt next booking: As a member, I can click to book the next voyage.

## Member Portal – Future Privileges

- US022 — See future labels: As a member, I see greyed/labelled “Available at Gold/Diamond.”

## Admin Access

- US023 — Admin login: As an admin, I can log in to the admin dashboard.

## Admin Quick Stats

- US024 — View key stats: As an admin, I can see total cards, active cards, and newest bookings.
- US025 — Open details: As an admin, I can open card list or booking details from stats.

## Admin – Card Management

- US026 — Create card serials: As an admin, I can create card serials.
- US027 — Assign owners: As an admin, I can assign card owners.
- US028 — Override status: As an admin, I can update card status manually.

## Admin – Member Management

- US029 — Manage members: As an admin, I can view and edit the member list.

## Admin – Booking Management

- US030 — Link bookings: As an admin, I can link bookings to card serial and booking type.

## Admin – Pricing Management

- US031 — Update pricing: As an admin, I can update charter pricing.

## Admin – News/Notices

- US032 — Manage notices: As an admin, I can create and edit news/notices.

## Admin – Recent Activity

- US033 — View activity: As an admin, I can view a 3-day activity timeline.

## Admin – Navigation Modules

- US034 — Navigate modules: As an admin, I can access Members/Benefits/Reports/Settings.
